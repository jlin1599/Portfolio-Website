# Portfolio Website

This repository contains the code for my personal portfolio website. It is designed to showcase my skills, projects, and experience. The website is built using HTML, CSS, and JavaScript, with a focus on responsive design for both desktop and mobile screens.

## Features

- **Responsive Navigation**: Includes a desktop navigation bar and a mobile-friendly hamburger menu.
- **Profile Section**: Displays my profile picture and brief introduction.
- **About Section**: Provides more details about me, my background, and experiences.
- **Experience Section**: Lists my work experiences with brief descriptions.
- **Project Section**: Showcases a variety of personal and professional projects.
- **Contact Section**: Allows visitors to get in touch via social media and email.

## Technologies Used

- **HTML**: Used to structure the content of the website.
- **CSS**: Styled the website with a clean and modern layout, incorporating flexbox for responsiveness.
- **Google Fonts**: Integrated the 'Poppins' font for a sleek and readable typography.
- **Responsive Design**: The layout is designed to adjust for both desktop and mobile devices.
